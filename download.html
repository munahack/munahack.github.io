Antes de instalar Muña! Necesitas tener instalado Apache, Php, MySQL y phpMyAdmin. Si estas bajo Windows puede usar Xampp y si estas bajo Linux puedes seguir esta guía de instalación. Luego de esto empezamos con la instalación.
Pasos para la instalación de Muña v0.9
1. Descargue la aplicación web (Download ZIP -> GitHub)
2. Descomprime el archivo en la carpeta /htdocs si utilizas xampp o /www si utilizas otro o si estas bajo linux
3. Ahora puedes probar desde tu navegador http://localhost/muna/
4. Debes restaurar la base de datos, el script se encuentra en la carpeta /muna/database/malditawebd.sql
5. Restaura la base de datos en tu servidor MySql, puedes usar PhpMyAdmin para éste trabajo (http://localhost/phpmyadmin)
6. Actualiza el archivo /muna/conexion.inc colocando la cuenta y password de tu servidor MySql
$servidor = "localhost";
$cuenta = "tuCuenta";
$password = "tuPassword";
$database = "malditawebd";
7. En hora buena!! En tu navegador entra a http://localhost/muna/ y ya puedes usar Muña y practicar a Hackear =)
